How depth is read. Each surface sinks to a depth set by its ranking metric on a logarithmic scale — the biggest float in the sunlit shallows, the quietest settle on the benthic floor. Two metrics, in order of preference:

Page views (when available) — an edge HTML document load (edgeResponseContentTypeName = "html"), sampling-adjusted, pulled from Cloudflare’s GraphQL analytics by scripts/fetch-traffic.mjs. Needs CLOUDFLARE_API_TOKEN to carry Zone Analytics: Read.

Commit impact (the current stand-in) — the number of git commits that have touched the files each endpoint deploys from, via scripts/fetch-commits.mjs reading deploy-registry.json’s pathspecs. No token required. Both are refreshed daily by the traffic-ingest workflow; path-based sub-pages roll up into their host surface, and only top-level surfaces are plotted.
