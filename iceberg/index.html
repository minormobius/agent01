The berg is a strange attractor. Its body is ~22,000 iterations of a de Jong map x'=sin(a·y)−cos(b·x); y'=sin(c·x)−cos(d·y), warped into an iceberg silhouette — so the ice has crystalline veining instead of flat fill, and each endpoint is scattered into it by the same chaotic sequence.

Depth = how much work the endpoint carries, on a log scale: the deepest-worked float near the sunlit tip, the lightly-touched sink toward the benthic keel. Two metrics, preferred in order:

Page views (when available) — sampling-adjusted edge HTML loads from Cloudflare, via scripts/fetch-traffic.mjs (needs Zone Analytics: Read).

Commit depth (the current metric) — every endpoint, sub-paths included, ranked by rename-aware commit count: scripts/fetch-commits.mjs unions git log --follow over each path's files, so work refactored in (the whole clock/ subtree was moved in, hiding ~87% of its commits from a naive count) still counts.
