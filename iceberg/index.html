How depth is read. Each surface sinks to a depth set by its page views on a logarithmic scale — the busiest sites float in the sunlit shallows, the quietest settle on the benthic floor. A “page view” is an edge HTML document load (edgeResponseContentTypeName = "html"), sampling-adjusted, pulled from Cloudflare’s GraphQL analytics by scripts/fetch-traffic.mjs and refreshed daily by the traffic-ingest workflow. Path-based sub-pages roll up into their host surface; only top-level surfaces are plotted.
