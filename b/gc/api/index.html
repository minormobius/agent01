gc API ⛒

block intelligence · read-only JSON

Four endpoints that answer block questions over public Bluesky data. No auth, CORS open, GET only — point any agent at it. Every input accepts a handle or a DID; every response is JSON with a source field telling you where the data came from: records (raw app.bsky.graph.block records, authoritative) or clearsky (the firehose-built block index at clearsky.services, used where the answer spans the whole network). Errors return { "error": "…" } with a 4xx/5xx status.
