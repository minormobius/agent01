gc ⛒

block checker · who blocks whom

Paste a set of accounts and a set of posters. gc reads each account's public block records and shows which of the posters they block — a tidy matrix of who has whom blocked. Read straight off the public network: no login, nothing written, nobody's permission needed. The same public block records you'll find on any account's PDS (and which dyad already surfaces between two accounts).

Public data only. This reads app.bsky.graph.block records, which every account stores publicly on its own PDS. It does not see private group-chat membership, and it does not catch blocks applied via a subscribed moderation block-list (those don't write a per-account block record) — so a "clear" here means no direct block. Enter the handles you want to check yourself.
