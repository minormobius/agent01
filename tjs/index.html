tjs

three.js workbenches — interactive 3D simulators with the physics taken seriously underneath the render.

  • DeckCompose a motion cell — place HBots + linear axes, mount them into a kinematic tree, export YAML
  • Motion suiteDrive any deck device — HBots + linear axes · jerk-limited motion · per-motor torque with mounted payload
Decks are agent-drivable: a local Claude can read a deck's capability manifest, write a sequence, and dry-run it against the same physics engine. More benches land here. Raw WebGPU experiments live at g.mino.mobi.
