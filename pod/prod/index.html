Prod · work in progress

The editing room. Load a com.minomobi.podcast.session manifest, pull every participant's chunked track into this one browser, align them on the shared epoch, drop in music, trim, and render the final mixdown.

What gets built here

  1. Fetch the session manifest → resolve each track's blob chunks across participants' PDSes.
  2. Multitrack timeline over the Web Audio API (lazy/streamed decode — full PCM won't fit in RAM).
  3. Coarse align by localStartOffsetMs; fine align by sync tone / measured clock skew.
  4. Music + bed tracks; gain, trim, crossfade.
  5. Render down with OfflineAudioContext (chunked), publish as com.minomobi.podcast.episode.

Track sync is the genuinely hard part — see the design notes in pod/README.md.
