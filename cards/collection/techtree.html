Tech Tree

An interactive map of 450 technologies spanning 3.3 million years of human invention, from stone tools to artificial general intelligence.

Layout

The tree uses polar coordinates with a parabolic funnel boundary (θ ∝ r²). Distance from the center is time — prehistoric origins sit near the hub, modern and AI-era technologies ring the outer edge. The funnel is narrow at the trunk (few root technologies) and widens quadratically into a broad canopy. Within the funnel, a 45° fan is divided into 12 domain sectors (Materials, Energy, Computing, Medicine, etc.); hex packing fills the available space at each radius.

Nodes

Each circle is a Wikipedia article. Nodes maintain a constant pixel size at any zoom level and hex-pack to avoid overlap when zoomed out. Wikipedia thumbnails load automatically. Border color indicates the technology's domain; legendary-rarity nodes glow.

Edges

Lines connecting nodes are prerequisite relationships — you need the parent technology before the child becomes possible. Edges connect the displayed hex windows directly, so they track the actual visual positions at any zoom level.

Interaction

  • Click a node to select it. Ancestors highlight gold, descendants highlight blue.
  • Scroll to zoom, drag to pan. Pinch-zoom works on touch.
  • The detail panel shows the Wikipedia extract, prerequisite chain depth, and links to navigate ancestors/descendants.

Data

Each technology has an era, domain, year, status (alive/dead/evolved), complexity (1–10), rarity, and a list of prerequisite technologies. The pool is curated for a “rebuild civilization” scenario — what would you need to reinvent if starting from scratch on another planet?
