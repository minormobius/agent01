The Ludographer

An endless catalogue of board games. Every number resolves — deterministically, on any machine, for ever — to the same complete game: a theme, a board, a tight set of mechanics that actually fit each other, the components they imply, a generated rulebook, an imaginary designer, and one deliberately broken rule.

№

No two adjacent numbers are alike; the catalogue does not end.

From the catalogue

How big is this, really?

—distinct game skeletons in the first 5,000 numbers
—mechanic primitives, tagged so only compatible ones combine
—board topologies (grid, hex, map, track, tiles, tableau, rondel, regions)
—thematic skins renaming the whole economy

A game here is a constraint walk: pick a core engine, satisfy what it needs with secondary mechanics that legally fit, derive the components, then choose a win condition the assembled set can actually be driven toward. Because the grammar only ever expresses combinations that cohere, every number you can open is a game that holds together — winnable by construction. (Catching the boring-but-legal games needs a bot that self-playtests each one; that's the next milestone, and it arrives with actually being able to play them.)

Kin to borges.mino.mobi, which does the same trick for medieval tales. The Ludographer is that apparatus re-aimed at the table.
