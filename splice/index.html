What this is

SpliceCraft (by Binomica Labs, MIT-licensed) is a terminal molecular-biology workbench — your whole cloning workflow in one tool. splice is an independent effort to bring that capability to the open web: re-author the deterministic core in Rust, compile it to WebAssembly, and reuse the original's hard C libraries (primer3, HMMER) rather than reinvent the science. On top of the tool sits a tiered game — because cloning is already a constraint-satisfaction puzzle, and the engine can grade any move objectively.

Today the full engine, four games, an interactive gel sim, and the end-to-end cloning campaign are live and playable above. The docs explain the approach; the roadmap shows where it goes next. The hardest technical risk — getting HMMER's hand-written SIMD core into WASM — has been spiked and proven.
