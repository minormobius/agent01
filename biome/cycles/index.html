biome · closed ecosystem for an infinite O'Neill cylinder

A non-spatial stocks-and-flows model of the cylinder interior as a living food web — fruit trees, ground crops and swamp reeds; pollinators that gate the fruit and the predators that eat them; living decomposers; and the crew. The question: does the loop close — does the web reach a steady state that feeds everyone and holds the air. Every flux is paired, so carbon/oxygen/water/nitrogen conserve to machine precision. Drag a knob; ~600 days re-run.  → Build your own web · Lake bioengine: fish + water treatment · Global web: land + lake in one box · Stability lab: will this web survive?
