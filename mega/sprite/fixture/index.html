Chamber Fixtures

Drawn set-dressing in the /yarrow hand — a genome → model → drawing, pure Canvas2D, seed-driven, feature density scaled to size. Where yarrow grows a milfoil stalk, this grows the rest of the furniture: hearths, anvils, looms, altars, barrels, shelves. The configuration implies variety (no two anvils alike) and the kind is tied to place — a forge gets anvils + a furnace, a temple an altar + a brazier — keyed to the same civic verbs the item engine uses. Built to vendor into hoop's ink.js beside drawStalk.

Every fixture, four seeds

The piece. fixtures.js is the library (14 detailed types: fixtureModel + draw + a zoom-gated detail pass), the FURNISH place→fixtures matrix, and furnishScene(role, rng, scene) — which places fixtures on the real foam floor from voronoi.js (vendored verbatim from hoop/paint, the same cells v3 paints). Node-tested (test/fixtures.selftest.mjs, 26 checks). The drop-in for hoop: vendor fixtures.js into ink.js and call furnishScene + drawFixture beside drawStalk over the chamber's existing paintCells.
