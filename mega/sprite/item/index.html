Item Genome

Every item is a tool — it does something. So its genotype is a position in a phylogeny (kingdom = the verb → phylum = the body-plan → species) plus a vector of orthogonal trait genes (durability, potency, mass, value, tech, ornament, complexity, provenance) that cut across the tree. Roll one from a hoard, splice two (cross-kingdom breeds chimeras), or mutate. Each is drawn as a map sprite (the body-plan) and an inventory glyph (the verb + signifiers). Built for /hoop; parallel to its civic genome.
