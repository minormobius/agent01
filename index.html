We are all web developers now.

For thirty years the open internet was technically accessible—anyone could view source, anyone could write HTML, anyone could register a domain and ship. In practice, the gap between “accessible” and “built” was filled by years of specialized training, mass-produced frameworks, and platform gatekeepers who offered convenience in exchange for ownership.

That gap closed. Claude Code is the hinge. Not because it writes code for you—but because it writes code with you, at the speed of intention. The person with the idea and the person with the implementation are now the same person, working in the same session. No handoff. No translation layer. No waiting.

What follows is what I’ve built since the gap closed. Static files on Cloudflare Pages. Data on the ATProto personal data server. No backend servers. No SaaS dependencies. Everything you see here was written in conversation—one human, one model, the open web.
